What is GFA Kaleidos?

a tool for pangenome graph statistics and analysis

Run GFA Kaleidos

Services

We do lots of cool things

  • Pangenome Graph Statistics

    Compute statistics for digraphs, bidirected graphs and biedged graphs to comprehensively characterize pangenome structures.

    Learn More
  • Multiple-GFA Comparison

    Support the integration and comparison of multiple GFA files, which is necessary for evaluating graph-building algorithms.

    Learn More
  • Offline Execution

    Support local execution and allows users to import results for web interaction.

    Learn More
  • Online Interaction

    Feature a user-friendly web interface with clear guidance for ease of use; Visual outputs can be exported to HTML or PDF formats.

    Learn More
